
Для корректной работы необходимо
================================

1. Подключить Raspberry Pi5 кабелем Ethernet к локальной сети, а Wi-Fi на нем лучше
выключить, так как одноплатник будет переведен в режим Wi-Fi хотспот с целью подключения
к нему вашего компьютера или телефона для настройки девайса в полевых условиях.

2. Установить на линукс пакет PHP любой версии, например:
    sudo apt install php

3. В файле php.ini (/etc/php.ini или /etc/php[VERSION]/php.ini) указать параметры:
    post_max_size = 800M
    upload_max_filesize = 800M

найти и раскомментировать строку ";extension=sockets" так, чтобы получилось:
    extension=sockets

4. Создать следующие поддиректории в файловой системе:
    sudo mkdir /srv
    sudo mkdir /srv/http    
    sudo mkdir /srv/http/player

каталогу "player" для простоты установить права на всё:
    sudo chmod 777 /srv/http/player

всем файлам с расширением "txt" для простоты установить доступ всем на чтение/запись:
    sudo chmod 666 /srv/http/*.txt

ВНИМАНИЕ: Никакие каталоги и файлы не стирать, даже пустые и с нулевым размером!!!

5. Установить и запустить apache или любой другой веб-сервер, например так:
    sudo apt install apache
    sudo systemctl enable httpd
    sudo systemctl start httpd

6. Запустите любой браузер и введите в строке адреса IP или имя этого вашего сервера, в
открывшейся страничке введите:
    логин: admin
    пароль: Panorama

после входа, эти и другие параметры можно изменить на ваше усмотрение.

7. Загрузить туда нужное панорамное видео в формате UHD/4K (разрешение 3840x2160, кодек HEVC)
и можно приступать к калибровке девайса с этим видео относительно окружающей местности.

8. Настроить остальные параметры если это требуется.
Установки Wi-Fi SSID и Wi-Fi пароль задают режим точки доступа, если девайсов несколько и
они расположены недалеко друг от друга, то надо на каждом устройстве задать разные SSID,
что-бы можно было подключаться веб-браузером к определенному девайсу, например для загрузки
нового видео. Все загруженные видео-файлы сохраняются, можно выбрать какой из них будет
активным, ненужные можно удалить. Новое видео воспроизводится сразу после его загрузки.
